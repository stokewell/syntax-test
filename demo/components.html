These components are built with the Web Components API, requiring no frameworks or libraries.

Responsive Image Component
A beautiful landscape image with 16:9 aspect ratio and lazy loading
<responsive-image src="https://picsum.photos/id/20/800/400" alt="Sample landscape" aspect-ratio="16:9" lazy> <span slot="caption">Caption text here</span> </responsive-image>
Card Component

This is a simple card component with title and content.

This card has an image and is clickable. It also uses shadow level 2.
<custom-card title="Card Title" shadow-level="1" clickable> <p>Card content goes here.</p> <div slot="footer"> <button class="btn-outline-sm">Button</button> </div> </custom-card>
