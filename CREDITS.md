# Credits & Acknowledgments

This project uses resources from several amazing open-source projects. We're grateful for the work of these contributors and want to properly acknowledge them.

## Framework Foundation

- **[Base Framework](https://github.com/stokewell/base)** - The lightweight foundation upon which Syntax is built (MIT License)
  - Base provides the clean, minimalist architecture that Syntax extends
  - Original author: [Stokewell](https://github.com/stokewell)
  - Syntax adds enhanced components, improved navigation, refined focus styles, and typography enhancements while preserving Base's performance-first approach

## Icon Libraries
